# Track Specification: Core Foundation

## Overview
This track focuses on establishing the base architecture for NoApp, including the Vite/React environment, Tailwind CSS integration, and the IndexedDB schema via Dexie.js.

## Scope
- Project scaffolding and dependency installation.
- Database layer implementation (Dexie.js).
- Global state management (Context API) for theme and active views.
- Core UI Layout (Sidebar + Editor).
- Basic CRUD for Folders and Notes.

## Technical Requirements
- Use Vite for build.
- Use Dexie.js for storage.
- Use Tailwind CSS for the 'papery' aesthetic.
